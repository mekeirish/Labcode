# 🧪 Labcode

> Une Progressive Web App (PWA) conçue comme un bac à sable (sandbox) mobile d'expérimentation pour le développement et le prototypage rapide.

Développée selon le protocole **PWA-CORE**, **Labcode** centralise l'intégralité de sa structure, de ses styles et de ses scripts dans une architecture monolithique. Idéal pour le "vibe coding" et l'itération ultra-rapide assistée par IA, ce laboratoire de poche permet de tester des composants isolés, des interfaces réactives et des comportements complexes directement sur mobile, sans friction.

---

## ✨ Fonctionnalités Clés

- 📱 **Sandbox Mobile-First** : Un environnement de test léger, réactif et optimisé pour les écrans tactiles.
- 🎨 **UI Immersive & Fluide** : Interface soignée adoptant des codes esthétiques skeuomorphiques et neumorphiques, avec textures réalistes et gestion native du mode sombre.
- ⚡ **Architecture Monolithique** : Zéro configuration, pas de dépendances lourdes, tout s'exécute instantanément depuis un fichier unique (`index.html`).
- 🔋 **PWA Autonome** : Entièrement installable sur iOS, Android et Desktop avec support complet du mode hors-ligne via Service Workers.

## 🛠️ Stack Technique

- **Frontend** : HTML5, CSS3 (Effets 3D, variables dynamiques, grilles responsives)
- **Logic & Core** : JavaScript ES6+ (Vanilla JS, gestion d'état réactive)
- **PWA** : Web App Manifest, Service Workers, LocalStorage
- **Philosophie de dév** : Vibe Coding & PWA-CORE Architecture

## 🚀 Installation & Utilisation Rapide

L'architecture monolithique élimine le besoin d'outils de build complexes (`npm`, `bundlers`, etc.).

1. **Récupérer le projet** :
Télécharge ou clone ce dépôt GitHub directement sur ton espace de travail local.

2. **Lancer le Lab** :
Ouvre simplement le fichier `index.html` dans ton navigateur, ou utilise une extension de serveur local (comme *Live Server* sur VS Code).

3. **Installer la PWA** :
Clique sur l'icône d'installation dans la barre d'adresse (Chrome/Edge) ou utilise l'option "Sur l'écran d'accueil" (Safari sur iOS) pour l'avoir comme une app native.

## 📐 Architecture PWA-CORE

Ce projet respecte strictement le modèle du fichier unique (`index.html`), offrant :
- Une **lisibilité maximale** pour les agents et assistants IA, facilitant les injections de code et les modifications de fonctionnalités en un seul prompt.
- Un **déploiement instantané** sans étape de compilation.
- Une **liaison directe** entre les styles graphiques et la logique applicative.
